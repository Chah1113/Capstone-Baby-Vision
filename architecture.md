## 구성 요소 설명

### 1. 테스트 PC
- 웹캠을 연결하여 영상 입력을 받음
- `main.py` 브릿지 앱 실행
- RTSP 프로토콜을 통해 서버로 영상 스트리밍 전송

---

### 2. 우리 서버

#### MediaMTX
- RTSP 스트림 수신 및 관리
- 영상 스트림을 내부 처리 파이프라인으로 전달

#### YOLOv8 워커
- 실시간 영상 분석 (객체 탐지)
- 감지 결과 생성

#### DB
- 분석 결과 저장
- 이벤트 로그 관리

#### Flutter 앱
- 특정 이벤트 발생 시 알림 전송
- 사용자에게 실시간 상황 전달

---

## 데이터 흐름 요약

1. 웹캠 → 영상 캡처  
2. 브릿지 앱 → RTSP로 서버 전송  
3. MediaMTX → 스트림 수신  
4. YOLOv8 → 객체 탐지  
5. DB → 결과 저장  
6. Flutter 앱 → 사용자 알림  

---

## 핵심 포인트

- RTSP 기반 실시간 스트리밍
- AI 객체 탐지 자동화 (YOLOv8)
- 이벤트 기반 알림 시스템
